Starting with icons

Bruno icons are SVGs (Scalable Vector Graphics), so you can use SVG to import. Note: No downloading or instalitaion is required.

Use the following to import Bruno SVGs:

      <svg width="width" height="height">
        <use xlink:href="https://brunozhon.github.io/Icons/iconname.svg" />
      </svg>

Or you can use img tag:

      <img src="https://brunozhon.github.io/Icons/iconname.svg" role="img" />

Why role="img"

The role="img" is required because browsers inconsistently announce SVG's in img tags with voice assictence. Learn more in simplyaccesible.com

Since the "S" part of SVG means "Scalable", you can add the meta tag without resizing problems:

      <meta name="viewport" content="width=device-width, initial-scale=1.0" /;>
