JSON

How JSON works

1. JavaScript object

Pretend you need to transfe this data to a server:

{name:"John Doe", age: 30, ancestors:["Jane Doe", "Seborah Doe"]}

You can use the JSON.stringfy() function. It will convert the object into a JSON string

2. Send to the server and response

Now the object looks like this: '{"name":"John Doe", "age":30, "ancestors":["Jane Doe", "Seborah Doe"]}'

The server will give you a response, such as "Hello, John Doe. I want to visit your ancestors: Jane Doe and Seborah Doe", but we will just focus on the server returning the object

Turn it into JSON

Now, you can use the JSON.parse() function to parse it into JSON.

The result:

{"name":"John Doe", "age":30, "ancestors":["Jane Doe", "Seborah Doe"]}
