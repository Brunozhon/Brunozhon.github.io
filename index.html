HTML is one of the programming languages used for web development. It is used to display my website. Let's start learning HTML!

The first things to put in the document are <!DOCTYPE html>, <html>, and<body>. Let's start coding! Name your file index.html and copy this code:

<!DOCTYPE html>
<html>
  <body>
    <p>Hello world!<p>
  </body>
</html>

This will display:

Hello world!

Note: We used Bootstrap to style our text. It also styles the results. If you want to style your pages, check out the Bootstrap documatation when you have mastered HTML and maybe CSS or JavaScript!

I suggest you to add a title. Where do you add a title? Check it out! (I will highlight added code in green and deleted code in red.)

<!DOCTYPE html>
<html>
  <head>
    <title>Hello world!</title>
  </head>
  <body>
    <p>Hello world!<p>
  </body>
</html>

The result will be the same but there will be a title (if you view in in a browser).

We are also going to learn about the line break. Although we're not going to add it, it is a important element to learn. It looks like this: <br />.

Here is a thing to help you better understand it.

This text is on the top of the line break.
This text is on the bottom of the line break.

Here is the source code:

<span style="background: red; color: white;">This text is on the top of the line break.</span>
<br />
<span style="background: red; color: white;">This text is on the bottom of the line break.</span>
